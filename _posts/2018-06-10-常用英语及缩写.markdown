#### 常用缩写 
    A
    average ——> avg

    B
    back ——> bk
    background ——> bg
    break ——> brk
    buffer ——> buf

    C
    color ——> cr(clr)
    control ——> ctrl

    D
    data ——> dat
    delete ——> del
    document ——> doc

    E
    edit ——> edt
    error ——> err
    escape ——> esc

    F
    flag ——> flg
    form ——> frm

    G
    grid ——> grd

    I
    increment ——> inc
    information ——> info
    initial ——> init
    insert ——> ins
    image ——> img

    L
    label ——> lab
    length ——> len
    list ——> lst
    library ——> lib

    M
    manager ——> mngr(mgr)
    message ——> msg

    O
    Oracle ——> Ora

    P
    panorama ——> pano
    password ——> pwd
    picture ——> pic
    point ——> pt
    position ——> pos
    print ——> prn
    program ——> prg

    S
    server ——> srv
    source ——> src
    statistic ——> stat
    string ——> str
    Sybase ——> Syb

    T
    temp ——> tmp
    text ——> txt

    U
    user ——> usr

    W
    window ——> wnd(win)
